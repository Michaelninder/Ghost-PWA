# Social Media Ghost Apps (PWA)

A dual-app Progressive Web App (PWA) repository designed to simulate the interface of TikTok and Facebook, returning a persistent **"Service Unavailable"** error. 

Intended as a **Digital Safety Tool** to provide positive friction for vulnerable users, breaking the habit loop of addictive and high-risk apps.

## 📂 Repository Structure
- `/index.html` - Root directory page for easy navigation.
- `/tiktok/*` - Isolated PWA for TikTok.
- `/facebook/*` - Isolated PWA for Facebook.

## 🚀 Setup & Deployment

1. **Host on GitHub Pages:**
   - Fork or upload this code to a new GitHub repository.
   - Go to **Settings > Pages**.
   - Under **Build and deployment**, set source to `Deploy from a branch` and select `main`.
   - Your site will publish to `https://<your-username>.github.io/<repo-name>/`.

2. **Installation on Target Device:**
   - **For TikTok:** Open `https://<your-username>.github.io/<repo-name>/tiktok/` on the phone.
   - **For Facebook:** Open `https://<your-username>.github.io/<repo-name>/facebook/` on the phone.
   - **iOS (Safari):** Tap *Share* > *Add to Home Screen*.
   - **Android (Chrome):** Tap *Three Dots* > *Add to Home Screen*.

3. **Deploy the Decoy:**
   - Delete or restrict the actual TikTok/Facebook apps using screen time controls.
   - Place these newly installed "Ghost App" icons in their exact place on the home screen.
